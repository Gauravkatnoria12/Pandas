# Pandas Learning Repository

This repository contains my learning progress and practice exercises for the Python `pandas` library.

## Contents

- `Pandas.ipynb` - A notebook for general Pandas exploration and examples.
- `DataFrames.ipynb` - Examples and exercises working with Pandas DataFrames.
- `Missing_Data.ipynb` - Notes and practice on handling missing data.
- `Concat&Subset&ConditionalRender.ipynb` - Examples on concatenation, subsetting, and conditional rendering.
- `pokemon.csv` - Sample dataset used for Pandas analysis.

## Purpose

The goal of this repo is to track and organize hands-on learning with Pandas, including data manipulation, cleaning, filtering, and visualization techniques.

## How to Use

1. Open the notebooks in Jupyter Notebook or JupyterLab.
2. Run the cells sequentially to explore the code and outputs.
3. Modify or extend the examples to deepen your understanding of Pandas.

## Notes

- The notebooks are practical learning resources and may include step-by-step examples.
- `pokemon.csv` can be used as a sample dataset for practicing Pandas operations.
